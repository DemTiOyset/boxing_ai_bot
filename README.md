# boxing_ai_bot

## This is an AI Telegram bot designed to answer questions about boxing.

### If you liked my project and want to import it, then you need to:
### 1. You need to create a settings.py file and create the TELEGRAM_TOKEN and OPEN_API_KEY variables

#### 2. Register the BotFather bot and receive a token, which you must insert into the values ​​of the TELEGRAM_TOKEN variable in the settings.py file

#### 3. Get a token from Open AI by the instruction [Open AI](https://platform.openai.com/docs/quickstart),  which you must insert into the values ​​of the OPEN_API_KEY variable in the settings.py file
